# 浏览器缓存

# 浏览器缓存的作用是什么？

  1. 缓存可以减少冗余的数据传输。节省了网络宽带，从而可以更快的加载页面。
  2. 缓存降低了服务器的要求，从而服务器更快的响应。

# 缓存规则

强缓存和协商缓存

  1. 强缓存：如果资源没过期，就读取缓存，如果过期了，则请求服务器。
