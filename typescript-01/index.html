# type 和 interface 区别

# 相同点

  1. 都可以描述一个对象或者函数

interface

interface User {
  name: string
  age: number
}

interface SetUser {
  (name: string, age: number): void;
}

  1. 两者都可以实现继承

# 不同点

  1. 语法不同
interface Point{
    x: number,
    y: number
}
type Point = {
    x: number,
    y: number,
}

interface setPoint {
    (x: number, y: number): void
}
type setPoint2 = (x: number, y: number) => void;

  1. interface 可以实现接口的 merge（interface可以定义多次，并将被视为单个接口），但是 type 不行(type 重复定义会报错)
interface Point {
    x: number
}
interface Point {
    y: number
}
var p:Point = {
    x: 1,
    y: 2
}

  1. Type可以用于更多的类型，例如：可以声明基本类型别名、联合类型、元组等类型
// 基本类型别名
type Name = string

// object
type ObjA = {a: number};
type ObjB = {b: number};

// 联合类型
type UnionObj = ObjA | ObjB;

// 元组类型
type Arr = [number, string];

// 当你想获取一个变量的类型时，使用 typeof
var div = document.creatElement('div');
type Div = typeof div;
